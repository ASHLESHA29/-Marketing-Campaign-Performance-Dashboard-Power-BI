# 📢Marketing-Campaign-Performance-Dashboard-Power-BI
This project showcases a Marketing Campaign Performance Dashboard built using Power BI to analyze digital advertising campaigns across multiple platforms and regions.

The dashboard helps evaluate:
Campaign effectiveness
Advertising spend vs revenue
ROI and conversion efficiency
Platform-wise and region-wise performance

## 🎯 Business Objectives

Track Total Revenue and Total Ad Spend
Measure key marketing KPIs:
ROI
CPC (Cost per Click)
CTR (Click Through Rate)
CPA (Cost per Acquisition)
Compare performance across:
Platforms (Facebook, Google, Instagram, LinkedIn)

## 🧠 Key Insights

Instagram and LinkedIn show the highest ROI among platforms.
Facebook accounts for the highest ad spend.
Europe generates the highest number of conversions.
Revenue peaks during later months, indicating campaign optimization over time.
